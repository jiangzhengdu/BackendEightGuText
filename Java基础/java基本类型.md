数据类型
===

数值型 ： 整数类型
byte[1]   一个字节  -128～127
short[2]  两个字节  -(2^15) ~ 2^15-1
int[4]   four bytes   -2^31 ~ 2^31-1
long[8]  eight bytes  -2^63 ~ 2^63-1  
         浮点类型 float[ 4]  double[8]

字符型   char[2]  'a'
布尔型   boolean[1]  true  false
